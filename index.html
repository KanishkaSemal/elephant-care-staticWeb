Project Scope

Literature Survey

Recent advancements in IoT and machine learning have improved wildlife conservation. Yet, real-time health monitoring for large animals like elephants remains a challenge. While early methods like GPS collars only tracked location, newer systems integrate biometric sensors like PPG to gather physiological data. Our system overcomes previous limitations such as EMI issues, weak transmission, and limited data accuracy. We use improved LoRa transmission, AI-based gait analysis, and ethical, non-invasive designs to ensure effective and scalable elephant monitoring.

Research Gap

Existing solutions often focus on individual aspects like GPS or motion detection, lacking integration. There's a gap in combining biometric sensors, robust long-range communication, and intelligent alerting systems into a single, wearable solution optimized for elephants in the wild.

Research Problem

How can a wearable IoT-enabled belt effectively monitor and analyze the biometric data of elephants in real-time to identify and respond to emergency situations, thereby enhancing their protection and overall health management?

Research Objectives

To design and implement an IoT-based solution aimed at protecting elephants using advanced detection, monitoring, and alerting technologies focused on real-time health monitoring.

  • Realtime data collection and transmission.
  • Develop durable belt for elephant.
  • Real-time health monitoring.
  • Emergency detection and notification.
  • Providing user-friendly interface.

Methodology

The "Elephant-Care" system consists of a wearable IoT belt and a real-time data processing system. The belt captures vital parameters like heart rate, SpO‚ÇÇ, temperature, and movement, then transmits it via LoRa to a base system. The dual-core ESP32 processes and sends the data, with emergency alerts triggered instantly. This robust, ethical system ensures minimal discomfort to elephants while enabling early emergency detection and conservation.

A. IoT-Enabled Smart Elephant Detection System

1) Sensors & Microcontroller: MAX30102 for SpO‚ÇÇ, LM75 for temperature, MPU6050 for motion detection, managed by ESP32 microcontroller.
2) Communication Protocol: LoRa 868 MHz, JSON packet encoding, CRC error-checking, and RTOS-based task scheduling for fast and reliable alerts.
